
## opis: aplikacja jest przeznaczona dla studentow z calego swiata która bedzie słuzyła do zapisywania waznych terminow (np. kolokwium, sesja, termin oddania pracy zaliczeniowej), notatek z wykladow oraz cwiczen z podzialem na dany przedmiot. Apliakcja bedzie mogła wysylac powiadomienia o zblizajacym sie terminie oddania pracy domowej lub nadchodzacyh zajeciach, zalozenie jest zeby komunikat pojawiaj sie w pasku powiadomiec na windowsie, telefonie z systemem ANDROID i iOSna zablokowanym ekranie. 

## Glowne funkcje aplikacji:
- tworzenie notatek
- zapisywanie waznych terminow
- zmiana jezyka wyswietlania
- mozliwosc edycji i usuwania notatek
- wklejanie zdjec i linkow
- podzial notatek na kategorie np: nazwa przedmiotu, kolokwium prace domowe
- wyszukowanie notatek za pomoca tematu lub nazwy

## Przykladowe kategorie notatek:
- laboratoria
- wyklady
- kolokwia
- cwiczenia
- wazne terminy
- projekty

 ## Przykladowe przedmioty:
- Lektorat z ang.
- matematyka
- paradygmaty
- informatyka

 ## Cel projektu:
Celem projektu jest stworzenie prostej aplikacji dla studnetow ktora ulatwia zarzadzanie notatkami w codziennej organizacji nauki oraz pokazanie umiejetnosci pracy z github
